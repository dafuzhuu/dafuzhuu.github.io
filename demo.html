手搓一个最简易博客

整体布局

风格是借鉴这篇博客，主要特点是大面积空白以及鼠标悬停时的按钮放大，需要自定义style如下。

字体设计

<style>
  body {
    font-size: 16px;
    font-family: 'Times New Roman', serif;
    margin-left: 350px;
    margin-top: 40px;
    line-height: 1.6; /* 行间距 */
  }
</style>

超链接设计

<style>
  a {
    color: inherit;
    text-decoration: none;
    transition: color 0.3s ease;
  }
  a:hover {
    color: red;
  }
</style>

返回主页键

<style>
  .home-button {
    position: absolute;
    top: 15px;
    right: 25px;
    display: inline-block;
    padding: 5px 10px;
    margin: 10px 0;
    background-color: white;
    color: black;
    text-align: center;
    text-decoration: none;
    font-weight: bold;
    border-radius: 5px;
    transition: transform 0.3s, background-color 0.3s;
  }
  .home-button:hover {
    color: inherit;
    transform: scale(1.1);
    background-color: lightgray;
  }
</style>

基础零件

虽然博客的框架和主题越来越丰富，但最常用的零件无外乎：列表、图片、表格、公式、代码块。

列表

有序列表

<ol>
  <li><a>item1</a></li>
  <li><a>item2</a></li>
</ol>
  1. item1
  2. item2

无序列表

<ul>
  <li><a>item1</a></li>
  <li><a>item2</a></li>
</ul>
  • item1
  • item2

图片


            <img src="favicon.png" alt="Pokeball" width="200" height="200">

表格

<table>
  <thead>
    <tr>
      <th>月份</th>
      <th>收入</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>一月</td>
      <td>$1000</td>
    </tr>
    <tr>
      <td>二月</td>
      <td>$800</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>总计</td>
      <td>$1800</td>
    </tr>
  </tfoot>
</table>
月份 收入
一月 $1000
二月 $800
总计 $1800

在头部引入Mathjax，之后便可用\(\LaTeX\)编辑数学公式

<script src="https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.1/MathJax.js?config=TeX-MML-AM_CHTML"></script>

示例：\(f\in C[a,b]\)

\[\lim_{x\rightarrow p}f'(\xi(x))=f'(\lim_{x\rightarrow p}\xi(x))=f'(p)\neq 0\]

左右尖括号<, >在html里是特殊字符，如果要正常显示需分别用&lt;和&gt;代替

<style>
pre {
  width: 650px; /* 设置固定宽度 */
  overflow: auto; /* 内容超出时显示滚动条 */
  white-space: pre-wrap; /* 如果需要的话，可以让长行换行 */
  word-wrap: break-word; /* 防止内容溢出 */
  /* 其他样式设置，如边框、背景色等 */
  border: 1px solid #ddd;
  background-color: #f4f4f4;
  padding: 10px;
}
            </style>

语法为

&lt;pre&gt;&lt;code class="language-html"&gt;&lt;/code&gt;&lt;/pre&gt;
