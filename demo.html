手搓一个最简易博客

整体布局

风格是借鉴这篇博客，主要特点是大面积空白以及鼠标悬停时的按钮放大，需要自定义style如下。

字体设计

<style>
  body {
    font-size: 16px;
    font-family: 'Times New Roman', serif;
    margin-left: 350px;
    margin-top: 40px;
    line-height: 1.6; /* 行间距 */
  }
</style>

超链接设计

<style>
  a {
    color: inherit;
    text-decoration: none;
    transition: color 0.3s ease;
  }
  a:hover {
    color: red;
  }
</style>

返回主页键

<style>
  .home-button {
    position: absolute;
    top: 15px;
    right: 25px;
    display: inline-block;
    padding: 5px 10px;
    margin: 10px 0;
    background-color: white;
    color: black;
    text-align: center;
    text-decoration: none;
    font-weight: bold;
    border-radius: 5px;
    transition: transform 0.3s, background-color 0.3s;
  }
  .home-button:hover {
    color: inherit;
    transform: scale(1.1);
    background-color: lightgray;
  }
</style>
