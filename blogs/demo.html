手搓一个最简易博客

作为一个非cs学生，我搞不懂version control和shell这些（mit missing semester有讲但我懒得听）， 用hugo框架和一些主题时遇到各种自己无法解决的问题，不想费时间在这上面了，做了这款纯html的网站，保证稳定能用就好。

当然有很多不足，比如每篇文章都要重复同样的<head>和<footer>，而且html语法也远没有md简便。不过也算是对文章数量的惩罚机制吧， 以免我掉进“为了写而写”的陷阱。

整体布局

风格是借鉴这篇博客，主要特点是大面积空白以及鼠标悬停时的按钮放大，需要自定义style如下。

字体设计

<style>
  body {
    font-size: 16px;
    font-family: 'Times New Roman', serif;
    margin-left: 26%;
    margin-right: 26%;
    margin-top: 40px;
    line-height: 1.6; /* 行间距 */
  }
</style>

超链接设计

<style>
  a {
    color: inherit;
    text-decoration: none;
    transition: color 0.3s ease;
  }
  a:hover {
    color: red;
  }
</style>

返回主页键

<style>
  .home-button {
    position: absolute;
    top: 15px;
    right: 25px;
    display: inline-block;
    padding: 5px 10px;
    margin: 10px 0;
    background-color: white;
    color: black;
    text-align: center;
    text-decoration: none;
    font-weight: bold;
    border-radius: 5px;
    transition: transform 0.3s, background-color 0.3s;
  }
  .home-button:hover {
    color: inherit;
    transform: scale(1.1);
    background-color: lightgray;
  }
</style>

接着，在<body>下，<header>和<main>之前，添加


  <a href="index.html" class="home-button">返回主页</a>

自定义网页图标

准备好图片，上传到网站仓库，命名为"favicon.png"，在头部（也就是&lt;head&gt;）加入下面这段代码

<link rel="icon" href="favicon.png" type="image/png">
<link rel="apple-touch-icon" sizes="180x180" href="favicon.png">

脚注

<style>
  footer {
    color: gray;
    text-align: center;
    padding: 20px 0;
  }
</style>

在<main>之后插入下面的代码

<footer>
  <div>2024 &copy; <a href="https://github.com/ultimate233/ultimate233.github.io">zzzzdf page</a></div>
</footer>

基础零件

虽然博客的框架和主题越来越丰富，但最常用的零件无外乎：列表、图片、表格、公式、代码块。

列表

有序列表

<ol>
  <li><a>item1</a></li>
  <li><a>item2</a></li>
</ol>
  1. item1
  2. item2

无序列表

<ul>
  <li><a>item1</a></li>
  <li><a>item2</a></li>
</ul>
  • item1
  • item2

图片

<div class="center">
  <img src="../static/img/image2.jpg" alt="Photo" width="150" height="150">
</div>

为了居中，定义类"center"如下

<style>
  .center {
      text-align: center;
  }
</style>

表格

<table>
  <thead>
    <tr>
      <th>月份</th>
      <th>收入</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>一月</td>
      <td>$1000</td>
    </tr>
    <tr>
      <td>二月</td>
      <td>$800</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>总计</td>
      <td>$1800</td>
    </tr>
  </tfoot>
</table>

代码块

左右尖括号<, >在html里是特殊字符，如果要正常显示需分别用&lt;和&gt;代替

我自定义的代码块设计如下

<style>
  pre {
    color: #000; /* 文字颜色 */
    background: #fff; /* 背景颜色 */
    border: 1px solid #000; /* 边框颜色 */
    box-shadow: 5px 5px 10px rgba(0,0,0,0.2); /* 立体阴影效果 */
    font-family: monospace; /* 等宽字体 */
    padding: 1em;
    border-radius: 4px; /* 轻微的圆角 */
    margin: 1em 0; /* 保证周围有空间 */
    overflow: auto; /* 内容超出时显示滚动条 */
    white-space: pre; /* 保留空格和换行 */
  }
</style>

如果还想让代码高亮，增强可读性，需要在头部导入相应的库

<link href="https://cdnjs.cloudflare.com/ajax/libs/prism/1.24.1/themes/prism.min.css" rel="stylesheet" />
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.24.1/prism.min.js"></script>

语法为

<pre><code class="language-html">
    <!--  我的代码  -->
</code></pre>
