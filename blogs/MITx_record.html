Data Analysis - 6.419x

  1. Review
    • Hypothesis testing (p-value, power, type I & II error, Fisher exact test, t-test, confidence interval)
    • Likelihood ratio test (\(L(x)\), Neyman-Pearson lemma, asymptotic \(\Lambda=-2\log(L(x))\), Wilks theorem)
    • Multiple testing
      • family-wise error rate[\(FWER=P(V\geqslant 1)\)], at least 1 false rejection
      • false discovery rate
        • \[ FDR=\mathbb{E}\bigg [\frac{V}{N_1}\bigg ] \]

        Recall \(N_1\) is the total number of discoveries made (the total number of null hypotheses rejected), and \(V\) is the number of false discoveries (the number of null hypotheses that were falsely rejected).

      • Bonferroni correction
        • \[ p^i<\frac{\alpha}{m} \]
      • Holm-Bonferroni
        • \[ p^i<\frac{\alpha}{m-(i-1)} \]
      • Benjamini-Hochberg

        • Reject \(H_0\) when

        \[mp^i\leqslant \alpha\]
